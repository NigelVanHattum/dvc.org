usage: dvc plots modify [-h] [-q | -v] [-t [<path>]] [-x <field>] [-y <field>]
                        [--no-header] [--title <text>] [--x-label <text>]
                        [--y-label <text>]
                        [--unset [<property> [<property> ...]]]
                        target

Modify display properties of plot metrics files.
Documentation: <https://man.dvc.org/plots/modify>

positional arguments:
  target                Metric file to set properties to

optional arguments:
  -h, --help            show this help message and exit
  -q, --quiet           Be quiet.
  -v, --verbose         Be verbose.
  -t [<path>], --template [<path>]
                        Special JSON or HTML schema file to inject with the
                        data. See <https://man.dvc.org/plots#plot-
                        templates>
  -x <field>            Field name for X axis.
  -y <field>            Field name for Y axis.
  --no-header           Provided CSV or TSV datafile does not have a header.
  --title <text>        Plot title.
  --x-label <text>      X axis label
  --y-label <text>      Y axis label
  --unset [<property> [<property> ...]]
                        Unset one or more display properties.
